# Weekly update template

Use this as the body of a pinned `Weekly updates` issue comment.

```md
## Week of YYYY-MM-DD

### Shipped

- #123 Short shipped item.

### In progress

- #124 Short in-progress item.

### Blocked

- #125 Blocked item — blocked on decision/access/input/review.

### Next

- #126 Candidate next item.

### Decisions

- #127 Decision made or decision still needed.
```

Rules:

- Keep it factual.
- Link issues.
- Do not include private or sensitive context.
- If nothing meaningful happened, say so plainly.
